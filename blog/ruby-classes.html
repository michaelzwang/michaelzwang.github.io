Ruby Classes

February 3, 2016

This week I have been learning a lot about Ruby classes and their uses. In this post, I am going to cover the basics of using classes in Ruby.

Basically, classes are descriptions of a category or type of thing. They are used to organize your code and they assign attributes and methods to all objects of the class.

Let's look at a simple example where I have created a Greeting class:

      class Greeting
        #methods and variables go here
      end

Now I am going to add an #initialize method to the class that accepts a name variable. When you create a new instance of the class, the #initialize method is run automatically and will create instance variables, designated by the @variable_name notation. Instance variables are useful because they can be used for every instance of the Greeting class, but will have a unique value each time.

      class Greeting
        def initialize(name)
          @name = name
        end
      end

Now I can add methods to my class. And if I create a new instance michael for the class, I can then call the instance method.

      class Greeting
        def initialize(name)
          @name = name
        end

        def english
          puts "Hello, my name is #{@name}!"
        end
      end
    

      michael = Greeting.new("Michael")  #=> this creates a new object
      michael.english
                    #=>"Hello, my name is Michael!"

I can continue adding methods and also create new objects of the class!

      class Greeting
        def initialize(name)
          @name = name
        end

        def english
          puts "Hello, my name is #{@name}."
        end

        def spanish
          puts "Hola, me llamo #{@name}."
        end

        def french
          puts "Bonjour, je m'appelle #{@name}."
        end
      end
    

      michael = Greeting.new("Michael")
      michael.english
                    #=>"Hello, my name is Michael."

      josh = Greeting.new("Josh")
      josh.spanish
                    #=>"Hola, me llamo Josh."

      kelly = Greeting.new("Kelly")
      kelly.french
                    #=>"Bonjour, je m'appelle Kelly."

Overall, this was an introduction to the basics of Ruby classes. It is important to remember that nothing will happen with the defined class until a new object is created and a method is called. Additionally, when you use the new keyword, to create an object/instance of the class, the initialize method will be run, usually in order to create instance variables.
