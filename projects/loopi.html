Loopi

Overview

Loopi.run is a web app created out of a desire to take the monotony out of running while also increasing efficiency. Input a desired distance for your run and your current location and Loopi will generate a random route so your run ends where it began.

For my final project at Dev Bootcamp, I worked with three of my classmates on Loopi over the span of one week. We started out by refining our idea, followed by discussing the target audience and outlining of the app's features.

Feature Design

We ultimately decided to use a mobile-first approach with a sleek and modern design. In order to convert users, we designed the landing page to be simple and streamlined. The user has two input options to customize their run and a large "Go" button to initialize our functionality.

Once the user presses "Go," the page will display via Google Maps the randomly generated route based on the input distance and location. By using the up and down arrows, the user can go back to re-submit a new query or view step-by-step walking directions for the generated route.

Technologies

By using Ruby on Rails and PostgreSQL, Loopi allows users to create a profile and save their favorite routes for future use.

The primary algorithm for the route generation was implemented through the use of the Geokit gem and Google Maps API. Using the input distance and location, we utilize Geokit to provide us with four geographical coordinates. These coordinates are sent to Google Maps, which then creates walking directions and displays a visual map of the generated route.

Additional features that can be implemented in the future are the provision of topographical information and options to avoid certain areas.
